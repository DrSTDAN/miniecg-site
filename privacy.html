Política de Privacidad – MiniECGC3 Última actualización: 2025-08-26 1. Responsable Navcraftinternacional (“nosotros”). Contacto: admin@navcraftinternacional.com 2. Qué datos trata la app - Bluetooth LE: la app escanea y se conecta a dispositivos BLE cercanos para recibir señales ECG. - En Android ≤ 11: se solicita “Ubicación” únicamente porque el sistema exige este permiso para escanear BLE. La app NO usa la ubicación con fines distintos ni recolecta tu posición. - No recolectamos datos personales, no creamos perfiles, ni enviamos información a servidores propios. 3. Finalidad del tratamiento - Mostrar en tiempo real la señal ECG desde tu dispositivo BLE y calcular métricas locales (p. ej., BPM). Todo se procesa en el dispositivo. 4. Almacenamiento y retención - Los datos de señal se procesan en memoria (RAM) y no se guardan persistente mente, salvo que el usuario lo solicite explícitamente (si en el futuro se añade función de guardar). 5. Compartición de datos - No compartimos datos con terceros. No se transmite información a servidores externos. 6. Seguridad - El intercambio con el dispositivo BLE se realiza mediante el stack Bluetooth del sistema. Recomendamos usar dispositivos BLE confiables. 7. Derechos del usuario - Puedes desinstalar la app en cualquier momento. Si en el futuro añadimos funciones de respaldo o nube, incluiremos mecanismos para solicitar acceso o borrado. 8. Menores - Esta app no está dirigida a menores. 9. Cambios de esta política - Publicaremos cambios aquí. El uso continuo tras cambios implica aceptación. 10. Contacto - Para dudas sobre privacidad: admin@navcraftinternacional.com
